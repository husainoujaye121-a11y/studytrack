# StudyTrack

A full-stack study session tracker built with Node.js, Express, MongoDB, and Vanilla JavaScript.

## Features
- User registration and login (JWT authentication)
- Log study sessions with course, duration, and notes
- View and delete study sessions
- Responsive design

## Tech Stack
- **Backend:** Node.js, Express.js, MongoDB (Mongoose)
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Auth:** bcryptjs, JSON Web Tokens

## Setup
1. Clone the repo
2. Run `npm install`
3. Create `.env` with `PORT`, `MONGO_URI`, and `JWT_SECRET`
4. Run `npm run dev`
